# 🚍 Smart Student Commute

A smart web application that helps students avoid crowded routes and travel efficiently using real-time data.

## Live Deployment:
https://smartstudentcommute.vercel.app/

---

## 📌 Problem

Students in urban areas face:
- Traffic congestion  
- Crowded routes  
- Delays and unsafe travel  

There is no student-focused solution that provides **real-time, crowd-aware route suggestions**.

---

## 💡 Solution

Smart Student Commute provides:
- Real-time route suggestions  
- Crowd/congestion insights  
- Faster and safer commute options  

Designed specifically for **students (Classes 9–12)**.

---

## 🚀 Features

- 🔍 Search destinations  
- 🗺️ Interactive map (OpenStreetMap + Leaflet)  
- 🧭 Route suggestions (OpenRouteService API)  
- 🚦 Traffic/Crowd indicators (Low / Medium / High)  
- 🔐 Firebase Authentication (Login/Signup)  
- 📱 Responsive and clean UI  

---

## 🛠️ Tech Stack

**Frontend:**
- React (Hooks)
- Tailwind CSS

**Backend (Services):**
- Firebase (Auth + Firestore)

**APIs:**
- OpenStreetMap (Map tiles)
- Leaflet (Map rendering)
- OpenRouteService (Routing)
